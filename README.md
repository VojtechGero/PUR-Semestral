# TaskFlow – jednoduchý správce úkolů

🔗 **Online verze aplikace:**  
https://vojtechgero.github.io/PUR-Semestral

---

## Popis

TaskFlow je jednoduchá webová aplikace pro správu úkolů vytvořená v čistém JavaScriptu (bez použití frameworků jako React apod.).

Aplikace umožňuje uživatelům vytvářet a spravovat vlastní úkoly, které jsou ukládány lokálně v prohlížeči.

---

## Technologie

- HTML
- CSS
- JavaScript (bez frameworků)
- localStorage

---

## Funkcionalita

- registrace a přihlášení uživatele
- vytváření úkolů
- nastavení priority
- označení úkolu jako hotový
- mazání úkolů
- ukládání dat pomocí `localStorage`

---

## Ukládání dat

Data jsou ukládána pomocí `localStorage`, takže:
- aplikace funguje bez backendu
- data jsou uložena pouze v konkrétním prohlížeči

---

## Vlastní input

Aplikace obsahuje vlastní input pro zadávání štítků:
- uživatel zadá text a potvrdí Enterem
- štítky se dynamicky přidávají a zobrazují
- lze je odebírat kliknutím

---

## Splnění zadání

- aplikace je vytvořena celá vlastním kódem
- je veřejně dostupná (GitHub Pages)
- obsahuje více vrstev (UI, logika, data)
- obsahuje vlastní input (štítky)
- autentifikuje uživatele
- ukládá data pomocí localStorage
